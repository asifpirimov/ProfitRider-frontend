# ProfitRider Frontend

React + Vite frontend for ProfitRider - A profit tracking application for delivery drivers.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS
- **State Management**: React Context
- **Routing**: React Router
- **HTTP Client**: Axios
- **Authentication**: Google OAuth 2.0

## Backend

Backend API: https://profitrider-api.onrender.com

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file:

```bash
VITE_API_URL=https://profitrider-api.onrender.com
```

## Features

- Google OAuth authentication
- Session tracking
- Profit calculations
- Credit system
- Responsive design
